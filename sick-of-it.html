Problem space

Indecisive audiophiles rejoice. Sick of It aims to combat decision fatigue when looking for new music to listen to.

The app asks the user to select a color and emoji that resonates with them at that particular moment.

Sick of It uses those data points to synthesize a playlist through Spotify's API.

Tech used

Sick of It uses React for the frontend along with Axios to make calls to Spotify's API.

Pain points

Spotify's API can be a bit finicky and initially the oauth workflow posed a few issues. However after building the correct GET and POST methods, I was able to completely solve for those problems.

The next biggest hurdle in getting Sick of It built was making a scoring algorithm to take the user's input and translate it into endpoints that could be used to make an API call to Spotify.

I am ultimately happy with the result, but there are a few improvements that could be made to the algorithm to yield better results from the query.

Next iteration
